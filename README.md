# 💻 Laptop Price Prediction System

A Machine Learning web application built using Python, Streamlit, Pandas, NumPy, and Scikit-Learn that predicts laptop prices based on hardware specifications such as processor, RAM, storage, GPU, display size, operating system, and warranty.

## 📌 Project Overview

The Laptop Price Prediction System allows users to select laptop specifications from dropdown menus and instantly receive an estimated laptop price generated by a trained Random Forest Regression model.

The application uses a dataset containing laptop specifications and prices and applies machine learning techniques to predict the expected market value.

---

## ⚠️ Important Note

This model was trained on a laptop dataset collected in **2021**.

Therefore:

- All predicted prices correspond to **2021 market conditions**.
- Actual laptop prices in today's market may differ due to inflation, technological advancements, and changing market trends.
- The prediction should be interpreted as an estimate based on **2021 laptop pricing trends**.

## 🧠 Machine Learning Model

**Algorithm Used:** Random Forest Regressor

### Why Random Forest?

- Handles nonlinear relationships effectively
- Reduces overfitting through ensemble learning
- Provides high prediction accuracy
- Works well with mixed categorical and numerical features
